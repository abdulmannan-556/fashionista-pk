# Fashionista PK

Fashionista PK is an **online fashion store** offering trendy apparel, footwear, and accessories in Pakistan. Built with **React, Vite, and TailwindCSS**, this project is fully responsive and includes features like a shopping cart, product listings, and individual product pages.
